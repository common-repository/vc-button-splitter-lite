The VC Button Splitter (Lite) add-on allows you to set up button split tests within Visual Composer.


== Description ==
With VC Button Splitter you can easily test buttons through the natural Visual Composer backend editor. This allows you full customization of labels, colors, styles and links without a new learning curve.

It is extremely easy to create split tests and VC Button Splitter is built to be used naturally within visual composer. There is no coding knowledge needed and it is ready to use with no set up other than activation!

Multiple button versions allow you to see what gets you more clicks and engages your visitors. (with Stats Addon or VC Splitter PRO)

You are missing out on clicks from visitors and VC Button Splitter will help you convert them to customers.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the \'Plugins\' screen in WordPress
3. Go to the VC Splitter tab to activate the plugin and you are ready to get started!

You will now see the element VC Splitter added to your element grid in Visual Composer, add this element and create split tests within it, each split test variation will be shown to your users independently.
